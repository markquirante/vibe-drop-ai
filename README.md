# Vibe Drop AI

Vibe Drop AI is an AI-assisted symbolic music generation project for creating RnB and lo-fi MIDI ideas.

The goal is to combine AI planning with deterministic music-theory rules and MIDI rendering. Instead of relying on AI to directly create final MIDI files, Vibe Drop AI uses AI to suggest musical ideas such as key, scale, tempo, chords, instruments, rhythm style, and melody constraints, then uses a rule-based engine to validate and render those ideas safely.

Primary use case:

- Generate chord progressions, melodies, and musical starting points as MIDI files.
- Keep outputs compatible with DAWs such as FL Studio.
- Support an iterative producer workflow where AI helps plan ideas, while deterministic code keeps the MIDI musically valid and editable.

The project is currently in an early proof-of-concept stage.
